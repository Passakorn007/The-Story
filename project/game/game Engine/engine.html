Game Engine คืออะไร

Game Engine คือซอฟต์แวร์หรือโปรแกรมสร้างเกม เพื่อช่วยให้นักพัฒนาเกมทำงานได้สะดวกมากขึ้น ซอฟต์แวร์เหล่านี้จะถูกสร้างจากจากภาษาโปรแกรมเมอร์ ได้แก่ C, C++, JAVA เกมจะต้องถูกสร้างผ่าน Game Engine เหล่านี้ก่อนนำออกมานำวางขายซึ่ง Game Engine แต่ละตัว จะรวบรวมชุดคำสั่งที่ทำหน้าที่แทบทุกอย่าง เช่น การ Rendering ภาพ 2D 3D physics engine sound scripting animation artificial intelligence และอื่นๆอีกมากมายให้กับนักพัฒนา Engine แต่ละตัวจะมี จุดเด่นแตกต่างกัน ดังนั้น เมื่อนักพัฒนาและค่ายเกมต่างๆ ต้องการจะพัฒนา จะต้องดูว่า Game Engine ตัวไหนเหมาะกับเกมที่จะพัฒนา Game Engine ในปัจจุบันมีหลากหลาย Game Engine มาก บางค่ายเกมอาจสร้างเพื่อใช้สร้างเกมให้กับค่ายของตนเอง Game Engine เหล่านี้จะถูกเรียกว่า “in-house engines” หรือ Game Engine สำหรับทีมงานภายในและในสังกัดเท่านั้น เช่น EA จะใช้ Frostbite Engine หรือ Capcom จะใช้ RE Engine เป็นต้น หรือบางค่ายสร้าง Game Engine ที่ได้เปิดให้ผู้ใช้งานทั่วไปได้ชื้อหรือดาวน์โหลดกันไปใช้ได้ตามคอมพิวเตอร์ที่บ้าน เปิดโอกาสให้เหล่านักพัฒนาเกมทางบ้านที่สร้างเกมที่ตัวเองจินตนาการเอาไว้ได้ ยกตัวอย่างเช่น Unreal Engine 4 Unity Construct และอีกมากมาย เป็นต้น


Unreal engine


Game Engine สำหรับการพัฒนาเกมจากทางค่าย Epic Games ที่ได้รับความนิยมอย่างมากในการทำเกมปัจจุบัน ด้วยการเป็น Engine ฟรี ที่มาพร้อมกับความสามารถในการใช้งานที่เหมาะกับนักพัฒนามือใหม่และผู้ชำนาญ สามารถสร้างเกมเพื่อที่จะนำไปลงหลากหลาย Platform เช่น PC Console Mobile และมีระบบ Blueprint ที่ทำให้นักพัฒนาสร้างเกมได้โดนที่ไม่ต้องเขียนโค้ด อีกทั้งยังสามารถปรับแต่งได้อย่างหลากหลายให้เข้ากับอุปกรณ์และกราฟิกในหลาย ๆ รูปแบบ ถึงอย่างไรก็ดี Unreal Engine แลกมากับส่วนแบ่ง 5% ของราคาเกมเมื่อวางจำหน่าย


RE Engine


RE Engine หรือชื่อเต็ม ๆ คือ Reach for the moon Engine ที่มีจุดเด่นในเรื่องของรายละเอียดของสิ่งของต่าง ๆ ไม่ว่าจะเป็นของประกอบฉาก ร่างกายของตัวละคร แสงเงาล้วนทำออกมาได้สมจริงรองรับ FXAA + TAA และ Shadow cache นอกจากนี้ยังสามารถที่จะ Output สัญญาณแบบ 4K และรองรับ VR อีกด้วย ซึ่ง Engine นี้เป็นของ Capcom และใช้ได้เฉพาะการพัฒนาเกมในเครือของ Capcom เท่านั้น
